Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando a tag <b> (não semântica).

Nesta frase temos um termo em destaque utilizando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico utilizando a tag <i> (não semântico).

Nesta frase temos um termo em itálico utilizando a tag <EM> (semântico).

Marca texto

Podemos usar também um marcador de texto utilizando a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Texto grande e pequeno

Estamos crindo um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido, para que possa ser lido, mas não considerado.

Texto insirido

Podemos marcar um texto como insirido para dar uma ênfase e indicar que ele foi insirido depois.

Existe também o subinhado com a tag <u> que é não semântica.
